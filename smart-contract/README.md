# Fairlance Smart Contract

EVM contract that enables the user of freelance platform to operate with honesty and transparency.

## Commands

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts
```
